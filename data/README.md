# NeuroPit `data/` directory

This folder holds curated, generated, and reference data that the
runtime needs but that is not source code.

## Tracked files

### `persona_priors.json`

Per driver persona threshold offsets, generated by
`scripts/compute_persona_priors.py` from real F1 telemetry. The
cognitive engine loads this file at startup and shifts the
population level persona thresholds (Panic, Aggressive, Defensive,
Flow) by the per driver offset. The full math lives in
`docs/COGNITIVE_METHODOLOGY.md` under "Per driver priors".

The shipped artifact is calibrated against the 2021 Abu Dhabi race
because that is the playback session the streamer ships with, so
the priors line up with the demo data out of the box. Regenerate
against another session by running:

```bash
python scripts/compute_persona_priors.py --year 2023 --event "Las Vegas"
```

Schema:

```json
{
  "schema_version": 1,
  "source": {"year": 2021, "event": "Abu Dhabi", "session": "R"},
  "documentation": "Population defaults (PersonaThresholds...)",
  "drivers": {
    "VER": {
      "panic_stress_offset": 0.71,
      "aggressive_stress_offset": 0.59,
      "defensive_confidence_offset": -0.47,
      "flow_confidence_offset": 0.35,
      "hot_envelope_z": 0.118,
      "proxy_medians": {
        "throttle_stdev": 41.826,
        "brake_stdev": 0.365,
        "mean_speed": 212.157
      }
    }
  }
}
```

Drivers missing from this file fall back to the population default
thresholds. A missing or malformed file is a soft degradation, not
a startup failure.

### `knowledge_sources/`

Curated motorsport literature compiled into the Qdrant ontology by
`src/backend/knowledge/docling_compiler.py` for IBM Granite to
ground reasoning against. The content is curated; the compilation
step is deterministic.

## Ignored subdirectories

`data/raw/` and `data/processed/` are git ignored. They are used as
scratch space by exploratory scripts and should not be committed.
